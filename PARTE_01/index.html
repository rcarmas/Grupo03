JS HTML DOM

"El Modelo de Objetos de Documento (DOM) del W3C es una plataforma y una interfaz de lenguaje neutral que permite que los programas y scripts accedan y actualicen dinámicamente el contenido, la estructura y el estilo de un documento". El estándar W3C DOM se divide en 3 partes diferentes: Core DOM: modelo estándar para todos los documentos XML DOM - modelo estándar para documentos XML HTML DOM - modelo estándar para documentos HTML
