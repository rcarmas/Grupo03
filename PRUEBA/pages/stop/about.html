Método jQuery stop()

El método jQuery stop()se utiliza para detener una animación o un efecto antes de que finalice.

El stop()método funciona para todas las funciones de efectos de jQuery, incluidas las animaciones deslizantes, desvanecidas y personalizadas.

Sintaxis:
$(selector).stop(stopAll,goToEnd);


El parámetro opcional stopAll especifica si también se debe borrar la cola de animación o no. El valor predeterminado es falso, lo que significa que solo se detendrá la animación activa, lo que permitirá que las animaciones en cola se realicen después.
El parámetro opcional goToEnd especifica si se completa o no la animación actual inmediatamente. El valor predeterminado es falso.
Entonces, de forma predeterminada, el stop()método elimina la animación actual que se está realizando en el elemento seleccionado.
